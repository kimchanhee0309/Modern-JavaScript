# 1. reduce ⭐️
* 배열을 하나의 값으로 축약하는 강력한 메서드

<details>
  <summary>(1) reduce 기본</summary>

  * **구조**
    ```JavaScript
    const result = array.reduce((누적값, 현재값, 인덱스, 배열) => {
        return 새로운 누적값;
    }, 초기값);
    ```
    
  * **기본 사용-합계**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    const sum = numbers.reduce((acc, num) => {
        return acc + num;
    }, 0);

    console.log(sum); // 15

    // 더 간단하게
    const sum2 = numbers.reduce((add, num) => acc + num, 0);
    ```
    
  * **동작 과정**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    numbers.reduce((acc, num) => {
        console.log(`누적: ${acc}, 현제: ${num}`);
        return acc + num;
    }, 0);

    // 출력:
    // 누적: 0, 현재: 1
    // 누적: 1, 현재: 2
    // 누적: 3, 현재: 3
    // 누적: 6, 현재: 4
    // 누적: 10, 현재: 5
    // 최종: 15
    ```
</details>

<details>
  <summary>(2) 다양한 활용</summary>

  * **곱셈**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    const product = numbers.reduce((acc, num) => acc * num, 1);
    console.log(product); // 120
    ```
    
  * **최댓값/최솟값**
    ```JavaScript
    const numbers = [5, 2, 8, 1, 9, 3];

    const max = numbers.reduce((max, num) => num > max ? num : max);
    // 초기값 설정이 없는 reduce의 초기값은 첫 번째 요소임
    console.log(max); // 9

    const min = numbers.reduce((min, num) => num < min ? num : min);
    console.log(min); // 1
    ```
    
  * **객체로 변환**
    ```JavaScript
    const users = [
        { id: 1, name: '홍길동' },
        { id: 2, name: '김철수' },
        { id: 3, name: '이영희' }
    ];

    // ID를 키로 하는 객체
    const usersById = users.reduce((acc, user) => {
        acc[user.id] = user;
        return acc;
    }, {});

    console.log(usersById);
    // {
    //    1: {id: 1, name: '홍길동'},
    //    2: {id: 2, name: '김철수'},
    //    3: {id: 3, name: '이영희'}
    // }
    ```
</details>

<details>
  <summary>(3) 그룹화</summary>

  * **카테고리별 그룹화**
    ```JavaScript
    const products = [
        { name: '노트북', category: '전자제품' },
        { name: '마우스', category: '전자제품' },
        { name: '책상', category: '가구' },
        { name: '의자', category: '가구' }
    ];

    const grouped = products.reduce((acc, product) => {
        const category = product.category;

        if (!acc[category]) {
            acc[category] = [];
        }

        acc[category].push(product);
        return acc;
    }, {});

    console.log(ground);
    // {
    //    '전자제품': [{name: '노트북', ...}, {name: '마우스', ...}],
    //    '가구': [{name: '책상', ...}, {name: '의자', ...}]
    // }
    ```
    
  * **통계 계산**
    ```JavaScript
    const students = [
        { name: '홍길동', score: 85 },
        { name: '김철수', score: 92 },
        { name: '이영희', score: 78 },
        { name: '박민수', score: 95 }
    ];

    const stats = students.reduce((acc, student) => {
        acc.total += student.score;
        acc.count += 1;
        acc.max = Math.max(acc.max, student.score);
        acc.min = Math.min(acc.min, student.score);
        return acc;
        }, { total: 0, count: 0, max: 0, min: Infinity });

    stats.average = stats.total / stats.count;

    console.log(stats);
    // {total: 350, count: 4, max: 95, min: 78, average: 87.5}
    ```
</details>

<details>
  <summary>(4) 중복 제거</summary>

  ```JavaScript
  const numbers = [1, 2, 2, 3, 3, 3, 4, 5, 5];

  const unique = numbers.reduce((acc, num) => {
      if (!acc.includes(num)) {
          acc.push(num);
      }
      return acc;
  }, []);

  console.log(unique); // [1, 2, 3, 4, 5]
  ```
</details>

<details>
  <summary>(5) 카운팅</summary>

  ```JavaScript
  const fruits = ['사과', '바나나', '사과', '오렌지', '바나나', '시과'];

  const count = fruits.reduce((acc, fruit) => {
      acc[fruit] = (acc[fruit] || 0) + 1;
      return acc;
  }, {});

  console.log(count);
  // {사과: 3, 바나나: 2, 오렌지: 1}
  ```
</details>

<details>
  <summary>(6) reduce vs 다른 메서드</summary>

  * **map으로 할 수 있는 것**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    // map
    const doubled1 = numbers.map(n => n * 2);

    // reduce로 구현
    const doubled2 = numbers.reduce((acc, n) => {
        acc.push(n * 2);
        return acc;
    }, []);
    ```
    
  * **filter로 할 수 있는 것**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    // filter
    const evens1 = numbers.filter(n => n % 2 === 0);

    // reduce로 구현
    const evens2 = numbers.reduce((acc, n) => {
        if (n % 2 === 0) {
            acc.push(n);
        }
        return acc;
    }, []);
    ```
    
  * **권장사항**
    ```JavaScript
    // 간단한 경우: map, filter 사용
    const doubled = numbers.map(n => n * 2);

    // 복잡한 집계: reduce 사용
    const grouped = items.reduce((acc, item) = {
        // 복잡한 로직...
    }, {});
    ```
</details>
