# 1. 삼항 연산자 ⭐️

<details>
  <summary>(1) 삼항 연산자란?</summary>

  * **기본 형태**
    ```
    조건 ? 참일 때 값 : 거짓일 때 값
    ```
    
  * **if-else와 비교**
    ```JavaScript
    // if-else
    let message;
    if (age >= 18) {
        message = '성인';
    } else {
        message = '미성년자';
    }

    // 삼항 연산자
    let message = age >= 18 ? '성인' : '미성년자';
    ```
    
</details>

<details>
  <summary>(2) 기본 사용법</summary>

  * **값 할당**
    ```JavaScript
    let age = 20;
    let status = age >= 18 ? '성인' : '미성년자';
    console.log(status); // 성인

    let score = 85;
    let grade = score >= 90 ? 'A' : 'B';
    console.log(grade); // B

    let isLoggedIn = true;
    let greeting = isLoggedIn ? '환영합니다' : '로그인하세요';
    console.log(greeting); // 환영합니다
    ```
    
  * **함수 반환값**
    ```JavaScript
    functioon getDiscount(isMember) {
        return isMember ? 0.1 : 0;
    }

    console.log(getDiscount(true));  // 0.1
    console.log(getDiscount(false)); // 0
    ```
    
  * **직접 출력**
    ```JavaScript
    let age = 15;
    console.log(age >= 18 ? '입장 가능' : '입장 불가');
    // 입장 불가
    ```
</details>

<details>
  <summary>(3) 중첩 삼항 연산자</summary>

  * **여러 조건**
    ```JavaScript
    let score = 85;

    let grade = score >= 90 ? 'A' :
                score >= 80 ? 'B' :
                score >= 70 ? 'C' :
                score >= 60 ? 'D' : 'F';

    console.log(grade);  // B
    ```
    
  * **주의: 가독성 저하**
    ```JavaScript
    // 나쁜 예(너무 복잡)
    let result = a > b ?
                  c > d ?
                    e > f ? 'x' : 'y'
                  : 'z'
                : 'w;

    // 좋은 예(if-else 사용)
    let result;
    if (a > b) {
        if (c > d) {
            result = e > f ? 'x' : 'y';
        } else {
            result = 'z';
        }
    }  else {
        result = 'w';
    }
    ```
</details>

<details>
  <summary>(4) 실전 활용</summary>

  * **기본값 설정**
    ```JavaScript
    function greet(name) {
        name = name ? name : '게스트';
        console.log('안녕하세요, ' + name + '님!');
    }

    // 더 간단하게(||와 비슷)
    function greet(name) {
        const userName = name || '게스트';
        console.log('안녕하세요, ' + userName + '님!');
    }
    ```
    
  * **조건부 클래스 이름**
    ```JavaScript
    const isActive = true;
    const className = isActive ? 'btn-active' : 'btn-inactive';

    // HTML에서 자주 사용
    const buttonClass = `btn ${isActive ? 'active' : 'inactive'}`;
    ```
    
  * **조건부 속성**
    ```JavaScript
    const user = {
        name: '홍길동',
        age: 25,
        role: isAdmin ? 'admin' : 'user',
        badge: isPremium ? 'premium' : null
    };
    ```
</details>

# 2. Spread 구문 ⭐️

<details>
  <summary>(1) Spread 구문이란?</summary>

  * **기본 형태**
    ```
    ...배열 또는 ...객체
    ```
    
  * **개념**
    ```JavaScript
    const numbers = [1, 2, 3];

    console.log(numbers);     // [1, 2, 3]
    console.log(...numbers);  // 1 2 3(펼쳐짐)
    ```
</details>

<details>
  <summary>(2) 배열 Spread</summary>

  * **배열 복사**
    ```JavaScript
    const original = [1, 2, 3];
    const copy = [...original];

    copy[0] = 100;

    console.log(original);  // [1, 2, 3] (원본 유지)
    console.log(copy);      // [100, 2, 3]
    ```
    
  * **배열 합치기**
    ```JavaScript
    const arr1 = [1, 2, 3];
    const arr2 = [4, 5, 6];

    // 기존 방식
    const merged1 = arr1.concat(arr2);

    // Spread 사용
    const merged2 = [...arr1, ...arr2];
    console.log(merged2); // [1, 2, 3, 4, 5, 6]

    // 중간에 요소 추가
    const merged3 = [...arr1, 'x', ...arr2];
    console.log(merged3); // [1, 2, 3, 'x', 4, 5, 6]
    ```
    
  * **배열에 요소 추가**
    ```JavaScript
    const numbers = [2, 3, 4];

    // 앞에 추가
    const withFirst = [1, ...numbers];
    console.log(withFirst); // [1, 2, 3, 4];

    // 뒤에 추가
    const withLast = [...numbers, 5];
    console.log(withLast); // [2, 3, 4, 5];

    // 양쪽에 추가
    const withBoth = [1, ...numbers, 5];
    console.log(withBoth); // [1, 2, 3, 4, 5];
    ```
    
  * **함수 인자로 전달**
    ```JavaScript
    const numbers = [1, 5, 3, 9, 2];

    // 레거시 방식
    console.log(Math.max.apply(null, numbers)); // 9

    // Spread 사용
    console.log(Math.max(...numbers));  // 9
    console.log(Math.min(...numbers));  // 1

    // 함수 예제
    function sum(a, b, c) {
        return a + b + c;
    }

    const nums = [1, 2, 3];
    console.log(sum(...nums)); // 6
    ```
</details>

<details>
  <summary>(3) 객체 Spread</summary>

  * **객체 복사**
    ```JavaScript
    const original = { name: '홍길동', age: 25 };
    const copy = { ...original };

    copy.age = 30;

    console.log(original.age); // 25(원본 유지)
    console.log(copy.age);  // 30
    ```
    
  * **객체 합치기**
    ```JavaScript
    const personal = { name: '홍길동', age: 25 };
    const contact = { email: 'hong@example.com', phone: '010-1234-5678' };

    const user = { ...personal, ...contact };
    console.log(user);
    // {
    //   name: '홍길동',
    //   age: 25,
    //   email: 'hong@example.com',
    //   phone: '010-1234-5678'
    // }
    ```
    
  * **속성 덮어쓰기**
    ```JavaScript
    const user = { name: '홍길동', age: 25, role: 'user' };

    // role만 변경
    const admin = { ...user, role: 'admin' };
    console.log(admin);
    // {name: '홍길동', age: 25, role: 'admin'}

    // 순서 중요! 뒤에 오는 것이 덮어씀
    const test1 = { ...user, age: 30 };
    console.log(test1.age); // 30

    const test2 = { age: 30, ...user };
    console.log(test2.age); // 25 (user의 age가 덮어씀)
    ```
    
  * **속성 추가**
    ```JavaScript
    const user = { name: '홍길동', age: 25 };

    const userWithId = { id: 1, ...user };
    console.log(userWithId);
    // {id: 1, name: '홍길동', age: 25}

    const userComplete = {
        ...user,
        email: 'hong@example.com',
        verified: true
    };
    ```
</details>
