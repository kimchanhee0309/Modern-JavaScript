# 1. 삼항 연산자 ⭐️

<details>
  <summary>(1) 삼항 연산자란?</summary>

  * **기본 형태**
    ```
    조건 ? 참일 때 값 : 거짓일 때 값
    ```
    
  * **if-else와 비교**
    ```JavaScript
    // if-else
    let message;
    if (age >= 18) {
        message = '성인';
    } else {
        message = '미성년자';
    }

    // 삼항 연산자
    let message = age >= 18 ? '성인' : '미성년자';
    ```
    
</details>

<details>
  <summary>(2) 기본 사용법</summary>

  * **값 할당**
    ```JavaScript
    let age = 20;
    let status = age >= 18 ? '성인' : '미성년자';
    console.log(status); // 성인

    let score = 85;
    let grade = score >= 90 ? 'A' : 'B';
    console.log(grade); // B

    let isLoggedIn = true;
    let greeting = isLoggedIn ? '환영합니다' : '로그인하세요';
    console.log(greeting); // 환영합니다
    ```
    
  * **함수 반환값**
    ```JavaScript
    functioon getDiscount(isMember) {
        return isMember ? 0.1 : 0;
    }

    console.log(getDiscount(true));  // 0.1
    console.log(getDiscount(false)); // 0
    ```
    
  * **직접 출력**
    ```JavaScript
    let age = 15;
    console.log(age >= 18 ? '입장 가능' : '입장 불가');
    // 입장 불가
    ```
</details>

<details>
  <summary>(3) 중첩 삼항 연산자</summary>

  * **여러 조건**
    ```JavaScript
    let score = 85;

    let grade = score >= 90 ? 'A' :
                score >= 80 ? 'B' :
                score >= 70 ? 'C' :
                score >= 60 ? 'D' : 'F';

    console.log(grade);  // B
    ```
    
  * **주의: 가독성 저하**
    ```JavaScript
    // 나쁜 예(너무 복잡)
    let result = a > b ?
                  c > d ?
                    e > f ? 'x' : 'y'
                  : 'z'
                : 'w;

    // 좋은 예(if-else 사용)
    let result;
    if (a > b) {
        if (c > d) {
            result = e > f ? 'x' : 'y';
        } else {
            result = 'z';
        }
    }  else {
        result = 'w';
    }
    ```
</details>

<details>
  <summary>(4) 실전 활용</summary>

  * **기본값 설정**
    ```JavaScript
    function greet(name) {
        name = name ? name : '게스트';
        console.log('안녕하세요, ' + name + '님!');
    }

    // 더 간단하게(||와 비슷)
    function greet(name) {
        const userName = name || '게스트';
        console.log('안녕하세요, ' + userName + '님!');
    }
    ```
    
  * **조건부 클래스 이름**
    ```JavaScript
    const isActive = true;
    const className = isActive ? 'btn-active' : 'btn-inactive';

    // HTML에서 자주 사용
    const buttonClass = `btn ${isActive ? 'active' : 'inactive'}`;
    ```
    
  * **조건부 속성**
    ```JavaScript
    const user = {
        name: '홍길동',
        age: 25,
        role: isAdmin ? 'admin' : 'user',
        badge: isPremium ? 'premium' : null
    };
    ```
</details>

# 2. Spread 구문 ⭐️

<details>
  <summary>(1) Spread 구문이란?</summary>

  * **기본 형태**
    ```
    ...배열 또는 ...객체
    ```
    
  * **개념**
    ```JavaScript
    const numbers = [1, 2, 3];

    console.log(numbers);     // [1, 2, 3]
    console.log(...numbers);  // 1 2 3(펼쳐짐)
    ```
</details>

<details>
  <summary>(2) 배열 Spread</summary>

  * **배열 복사**
    ```JavaScript
    const original = [1, 2, 3];
    const copy = [...original];

    copy[0] = 100;

    console.log(original);  // [1, 2, 3] (원본 유지)
    console.log(copy);      // [100, 2, 3]
    ```
    
  * **배열 합치기**
    ```JavaScript
    const arr1 = [1, 2, 3];
    const arr2 = [4, 5, 6];

    // 기존 방식
    const merged1 = arr1.concat(arr2);

    // Spread 사용
    const merged2 = [...arr1, ...arr2];
    console.log(merged2); // [1, 2, 3, 4, 5, 6]

    // 중간에 요소 추가
    const merged3 = [...arr1, 'x', ...arr2];
    console.log(merged3); // [1, 2, 3, 'x', 4, 5, 6]
    ```
    
  * **배열에 요소 추가**
    ```JavaScript
    const numbers = [2, 3, 4];

    // 앞에 추가
    const withFirst = [1, ...numbers];
    console.log(withFirst); // [1, 2, 3, 4];

    // 뒤에 추가
    const withLast = [...numbers, 5];
    console.log(withLast); // [2, 3, 4, 5];

    // 양쪽에 추가
    const withBoth = [1, ...numbers, 5];
    console.log(withBoth); // [1, 2, 3, 4, 5];
    ```
    
  * **함수 인자로 전달**
    ```JavaScript
    const numbers = [1, 5, 3, 9, 2];

    // 레거시 방식
    console.log(Math.max.apply(null, numbers)); // 9

    // Spread 사용
    console.log(Math.max(...numbers));  // 9
    console.log(Math.min(...numbers));  // 1

    // 함수 예제
    function sum(a, b, c) {
        return a + b + c;
    }

    const nums = [1, 2, 3];
    console.log(sum(...nums)); // 6
    ```
</details>

<details>
  <summary>(3) 객체 Spread</summary>

  * **객체 복사**
    ```JavaScript
    const original = { name: '홍길동', age: 25 };
    const copy = { ...original };

    copy.age = 30;

    console.log(original.age); // 25(원본 유지)
    console.log(copy.age);  // 30
    ```
    
  * **객체 합치기**
    ```JavaScript
    const personal = { name: '홍길동', age: 25 };
    const contact = { email: 'hong@example.com', phone: '010-1234-5678' };

    const user = { ...personal, ...contact };
    console.log(user);
    // {
    //   name: '홍길동',
    //   age: 25,
    //   email: 'hong@example.com',
    //   phone: '010-1234-5678'
    // }
    ```
    
  * **속성 덮어쓰기**
    ```JavaScript
    const user = { name: '홍길동', age: 25, role: 'user' };

    // role만 변경
    const admin = { ...user, role: 'admin' };
    console.log(admin);
    // {name: '홍길동', age: 25, role: 'admin'}

    // 순서 중요! 뒤에 오는 것이 덮어씀
    const test1 = { ...user, age: 30 };
    console.log(test1.age); // 30

    const test2 = { age: 30, ...user };
    console.log(test2.age); // 25 (user의 age가 덮어씀)
    ```
    
  * **속성 추가**
    ```JavaScript
    const user = { name: '홍길동', age: 25 };

    const userWithId = { id: 1, ...user };
    console.log(userWithId);
    // {id: 1, name: '홍길동', age: 25}

    const userComplete = {
        ...user,
        email: 'hong@example.com',
        verified: true
    };
    ```
</details>

# 3. 모던한 프로퍼티 표기법 ⭐️

<details>
  <summary>(1) 단축 프로퍼티(Shorthand Properties)</summary>

  * **기존 방식**
    ```JavaScript
    const name = '홍길동';
    const age = 25;

    const person = {
        name: name,
        age: age
    };
    ```
    
  * **단축 표기**
    ```JavaScript
    const name = '홍길동';
    const age = 25;

    const person = {
        name,  // name: name과 같음
        age    // age: age와 같음
    };

    console.log(person);  // {name: '홍길동', age: 25}
    ```
    
  * **실전 예제**
    ```JavaScript
    function createUser(name, age, email) {
        return {
            name,
            age,
            email,
            createdAt: new Data()
        };
    }

    const user = createUser('홍길동', 25, 'hong@example.com');
    console.log(user);
    // {
    //   name: '홍길동',
    //   age: 25,
    //   email: 'hong@example.com',
    //   createdAt: 2024-01-20T...
    // }
    ```
</details>

<details>
  <summary>(2) 계산된 프로퍼티 이름(Computed Property Names)</summary>

  * **동적 키 생성**
    ```JavaScript
    const key = 'name';
    const value = '홍길동';

    // 기존 방식
    const person1 = {};
    person1[key] = value;

    // 계산된 프로퍼티
    const person2 = {
        [key]: value
    };

    console.log(person2);  // {name: '홍길동'}
    ```
    
  * **실전 예제**
    ```JavaScript
    // 동적 객체 생성
    function createObject(key, value) {
        return {
            [key]: value
        };
    }

    console.log(createObject('name', '홍길동'));
    // {name: '홍길동'}

    console.log(createObject('age'. 25));
    // {age: 25}

    // 표현식도 사용 가능
    const prefix = 'user';
    const id = 123;

    const data = {
        [prefix + '_' + id]: '홍길동'
    };

    console.log(data);  // {user_123: '홍길동'}
    ```
</details>

<details>
  <summary>(3) 메서드 단축 표기</summary>

  * **기존 방식**
    ```JavaScript
    const person = {
        name: '홍길동',
        greet: function() {
            console.log('안녕하세요!');
        }
    };
    ```
    
  * **단축 표기**
    ```JavaScript
    const person = {
        name: '홍길동',
        greet() {
            console.log('안녕하세요!');
        },
        sayName() {
            console.log('제 이름은 ' + this.name);
        }
    };

    person.greet();    // 안녕하세요!
    person.sayName();  // 제 이름은 홍길동
    ```
    
  * **실전 예제**
    ```JavaScript
    const calculator = {
        value: 0,

        add(n) {
            this.value += n;
            return this;
        },

        subtract(n) {
            this.value -= n;
            return this;
        },

        multiply(n) {
            this.value *= n;
            return this;
        },

        getResult() {
            return this.value;
        }
    };

    const result = calculator
        .add(10)
        .multyply(2)
        .subtract(5)
        .getResult();

    console.log(result);  // 15
    ```
</details>

# 4. 옵셔널 체이닝(?.) ⭐️

* 안전하게 깊은 객체 속성에 접근하는 방법
 
<details>
  <summary>(1) 옵셔널 체이닝이란?</summary>

  * **문제 상황**
    ```JavaScript
    const user = {
        name: '홍길동',
        address: {
            city: '서울'
        }
    };

    // 정상 작동
    console.log(user.address.city);  // 서울

    // 문제 발생
    const user2 = {
        name: '김철수'
        // address 없음
    };

    console.log(user2.address.city);
    // TypeError: Cannot read Property 'city' of undefined
    ```
    
  * **기존 해결 방법**
    ```JavaScript
    // 방법 1: if문
    if (user2.address && user2.address.city) {
        console.log(user2.address.city);
    }
  
    // 방법 2: 삼항 연산자
    const city = user2.address ? user2.address.city : undefined;
    
    // 방법 3: 논리 연산자
    const city = user2.address && user2.address.city;
    ```
    
  * **옵셔널 체이닝 사용**
    ```JavaScript
    const city = user2.address?.city;
    console.log(city);  // undefined (에러 없음!)
    ```
</details>

<details>
  <summary>(2) 기본 사용법</summary>

  * **객체 속성**
    ```JavaScript
    const user = {
        name: '홍길동',
        address: {
            city: '서울',
            zipCode: '12345'
        }
    };

    console.log(user?.name);                   // 홍길동
    console.log(user?.address?.city);          // 서울
    console.log(user?.address?.country);       // undefined
    console.log(user?.contact?.email);         // undefined
    ```
    
  * **배열**
    ```JavaScript
    const users = [
        { name: '홍길동' },
        { name: '김철수' }
    ];

    console.log(users?.[0]?.name);    // 홍길동
    console.log(users?.[5]?.name);    // undefined

    const emptyArray = null;
    console.log(emptyArray?.[0]);     // undefined
    ```
    
  * **함수 호출**
    ```JavaScript
    const user = {
        name: '홍길동',
        greet: function() {
            return '안녕하세요!';
        }
    };

    console.log(user.greet?.());        // 안녕하세요!
    console.log(user.sayBye?.());       // undefined(에러 없음)

    // 실전 예제
    const data = {
        getName: function() {
            return '홍길동';
        }
    };

    const name1 = data.getName?.();      // 홍길동
    const nmae2 = data.getAge?.();       // undefined
    ```
</details>

<details>
  <summary>(3) 주의사항</summary>

  * **왼쪽 값만 체크**
    ```JavaScript
    const user null;

    // user가 null이므로 에러 없음
    console.log(user?.address.city);    // undefined

    // But, address가 있으면
    const user2 = { address: null };
    console.log(user2?.address.city);
    // TypeError! (address는 체크 안 됨)

    // 올바른 방법
    console.log(user2?.address?.city);  // undefined
    ```
    
  * **할당에는 사용 불가**
    ```JavaScript
    const user = {};

    // 에러
    // user?.name = '홍길동';

    // 올바른 방법
    if (user) {
        user.name = '홍길동';
    }
    ```
</details>

# 5. 배열 Destructuring(구조분해할당) ⭐️

<details>
  <summary>(1) 기본 Destructuring</summary>

  * **기존 방식**
    ```JavaScript
    const numbers = [1, 2, 3];

    const first = numbers[0];
    const second = numbers[1];
    const third = numbers[2];
    ```
    
  * **Destructuring**
    ```JavaScript
    const numbers = [1, 2, 3];

    const [first, second, third] = numbers;

    console.log(first);  // 1
    console.log(second); // 2
    console.log(third);  // 3
    ```
</details>

<details>
  <summary>(2) 다양한 패턴</summary>

  * **일부만 추출**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    const [first, second] = numbers;
    console.log(first, second); // 1, 2

    // 건너뛰기
    const [, , third] = numbers;
    console.log(third); // 3

    const [a, , , d) = numbers;
    console.log(a, d); // 1 4
    ```
    
  * **Rest 패턴**
    ```JavaScript
    const numbers = [1, 2, 3, 4, 5];

    const [first, ...rest] = numbers;
    console.log(first);  // 1
    console.log(rest);   // [2, 3, 4, 5];

    const [a, b, ...others] = numbers;
    console.log(a);      // 1
    console.log(b);      // 2
    console.log(others); // [3, 4, 5]
    ```
    
  * **기본값**
    ```JavaScript
    const numbers = [1];

    const [a, b = 0] = numbers;
    console.log(a);  // 1
    console.log(b);  // 0(기본값)

    const [x = 10, y = 20, z = 30] = [1, 2];
    console.log(x, y, z);  // 1 2 30
    ```
</details>

<details>
  <summary>(3) 활용 예시</summary>

  * **변수 교환**
    ```JavaScript
    let a = 1;
    let b = 2;

    // 기존 방식
    let temp = a;
    a = b;
    b = temp;

    // Destructuring
    [a, b] = [b, a];
    console.log(a, b); // 2 1
    ```
    
  * **함수 반환값**
    ```JavaScript
    function getMinMax(numbers) {
        return [Math.min(...numbers), Math.max(...numbers)];
    }

    const [min, max] = getMinMax([3, 1, 4, 1, 5, 9, 2, 6]);
    console.log(min, max); // 1 9
    ```
    
  * **문자열 분리**
    ```JavaScript
    const dateString = '2024-01-20';
    const [year, month, day] = dateString.split('-');

    console.log(year);   // 2024
    console.log(month);  // 01
    console.log(day);    // 20
    ```
    
  * **반복문**
    ```JavaScript
    const users = [
        ['홍길동', 25],
        ['김철수', 30],
        ['이영희', 28]
    ];

    for (const [name, age] of users) {
        console.log(`${name}: ${age}세₩);
    }

    // 홍길동: 25세
    // 김철수: 30세
    // 이영희: 28세
    ```
</details>

# 6. 객체 Destructuring(구조분해할당) ⭐️

<details>
  <summary>(1) 기본 Destructuring</summary>

  * **기존 방식**
    ```JavaScript
    const person = {
        name: '홍길동',
        age: 25,
        city: '서울'
    };

    const name = person.name;
    const age = person.age;
    const city = person.city;
    ```
    
  * **Destructuring**
    ```JavaScript
    const person = {
        name: '홍길동',
        age: 25,
        city: '서울'
    };

    const { name, age, city } = person;

    console.log(name);  // 홍길동
    console.log(age);   // 25
    console.log(city);  // 서울
    ```
</details>

<details>
  <summary>(2) 다양한 패턴</summary>

  * **순서 무관**
    ```JavaScript
    const person = { name: '홍길동', age: 25, city: '서울' };

    // 순서 상관없음
    const { city, name, age } = person;
    console.log(city, name, age);  // 서울 홍길동 25
    ```
    
  * **변수명 변경**
    ```JavaScript
    const person = { name: '홍길동', age: 25 };

    const { name: userName, age: userAge } = person;

    console.log(userName);  // 홍길동
    console.log(userAge);   // 25
    // console.log(name);   // ReferenceError
    ```
    
  * **기본값**
    ```JavaScript
    const person = { name: '홍길동' };

    const { name, age = 0, city = '미정' } =  person;

    console.log(name);  // 홍길동
    console.log(age);   // 0(기본값)
    console.log(city);  // 미정(기본값)
    ```
    
  * **변수명 변경 + 기본값**
    ```JavaScript
    const person = { name: '홍길동' };

    const { name: userName, age: userAge = 0 }
    = person;

    console.log(userName);  // 홍길동
    console.log(userAge);   // 0
    ```
    
  * **Rest 패턴**
    ```JavaScript
    const person = {
        name: '홍길동',
        age: 25,
        city: '서울',
        job: '개발자'
    };

    const { name, age, ...rest } = person;

    console.log(name);  // 홍길동
    console.log(age);   // 25
    console.log(rest);  // {city: '서울', job: '개발자'}
    ```
</details>

<details>
  <summary>(3) 중첩 Destructuring</summary>

  ```JavaScript
  const user = {
      name: '홍길동',
      age: 25,
      address: {
          city: '서울',
          zipCode: '12345'
      }
  };

  const {
      name,
      address:  { city, zipCode }
  } = user;

  console.log(name);      // 홍길동
  console.log(city);      // 서울
  console.log(zipCode);   // 12345
  // console.log(address);  // ReferenceError

  // address도 필요하면
  const {
      name: userName;
      address,
      address: { city: userCity }
  } = user;

  console.log(userName);    // 홍길동
  console.log(address);     // {city: '서울', zipCode: '12345}
  console.log(userCity);    // 서울
  ```
</details>
