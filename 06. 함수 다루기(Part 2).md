# 1. Arrow Function ⭐️⭐️

<details>
  <summary>(1) Arrow Function 기본 문법</summary>

  * **기존 함수 표현식**
    ```JavaScript
    const add = function(a, b) {
        return a + b;
    };
    ```
    
  * **Arrow Function**
    ```JavaScript
    const add = (a, b) => {
        return a + b;
    };
    ```
    
  * **더 간결하게**
    ```JavaScript
    // 한 줄일 때 중괄호와 return 생략 가능
    const add = (a, b) => a + b;
    console.log(add(10, 20));  // 30
    ```
</details>

<details>
  <summary>(2) 다양한 형태</summary>

  * **매개변수 없음**
    ```JavaScript
    // 괄호 필수
    const greet = () => {
        console.log('안녕하세요!');
    };

    // 한 줄
    const greet = () => console.log('안녕하세요!');
    greet();  // 안녕하세요!
    ```
    
  * **매개변수 1개**
    ```JavaScript
    // 괄호 생략 가능
    const double = x => x * 2;
    console.log(double(5));  // 10;

    // 괄호 써도 됨
    const triple = (x) => x * 3;
    ```
    
  * **매개변수 2개 이상**
    ```JavaScript
    // 괄호 필수
    const add = (a, b) => a + b;
    const multiply = (a, b) => a * b;
    ```
    
  * **여러 줄 코드**
    ```JavaScript
    const processUser = (user) => {
        const name = user.name.toUpperCase();
        const age = user.age + 1;
        return { name, age };
    };

    console.log(processUser({ name: 'hong', age: 25 }));
    // {name: 'HONG', age: 25}
    ```
    
  * **객체 반환**
    ```JavaScript
    // 중괄호가 코드 블록으로 인식됨
    const createUser = (name, age) => { name: name, age: age };

    // 괄호로 감싸기
    const createUser = (name, age) => ({ name: name, age: age });

    // 더 간결하게(ES6 단출 프로퍼티)
    const createUser = (name, age) => ({ name, age });

    console.log(createUser('홍길동', 25));
    // { name: '홍길동', age: 25 };
    ```
</details>

# 2. this의 이해

<details>
  <summary>(1) this란?</summary>

  * **this = 함수가 호출될 때 결정되는 특별한 값**
    ```JavaScript
    "나"라는 단어:
    - 홍길동이 말하면 -> 홍길동
    - 김철수가 말하면 -> 김철수

    this
    - 누가 호출하느냐에 따라 달라짐
    ```
</details>

<details>
  <summary>(2) this 결정 규칙</summary>

  * **1) 일반 함수 호출 - 전역 객체**
    ```JavaScript
    function showThis() {
        console.log(this);
    }

    showThis();
    // 브라우저: Window 객체
    // Node.js: global 객체
    ```
    
  * **2) 메서드 호출 - 메서드를 호출한 객체**
    ```JavaScript
    const person = {
        name: '홍길동',
        greet: function() {
            console.log('안녕하세요, ' + this.name);
        }
    };

    person.greet();  // 안녕하세요, 홍길동
    // this = person
    ```
    
  * **3) 생성자 함수 - 새로 생성된 객체**
    ```JavaScrpit
    function Person(name) {
        this.name = name;
    }

    const p1 = new Person('홍길동');
    console.log(p1.name);  // 홍길동
    // this = 새로 만들어진 객체
    ```
    
  * **4) call, apply, bind - 명시적으로 지정**
    ```JavaScript
    function greet() {
        console.log('안녕하세요, ' + this.name);
    }

    const person = { name: '홍길동' };

    greet.call(person);  // 안녕하세요, 홍길동
    // this.person
    ```
</details>

<details>
  <summary>(3) Arrow Function의 this</summary>

  * **Arrow Function은 this를 바인딩하지 않음**
    ```JavaScript
    const person = {
        name: '홍길동',
        greet1: function() {
            console.log(this.name);
        },
        greet2: () => {
            console.log(this.name);
        }
    };

    person.greet1();  // 홍길동(this = person)
    person.greet2();  // undefined (this = 전격 객체)
    ```
    
  * **Arrow Function은 상위 스코프의 this 사용**
    ```JavaScript
    const person = {
        name: '홍길동',
        friends: ['김철수', '이영희'],
        printFriends: function() {
            // 일반 함수
            this.friends.forEach(function(friend) {
                console.log(this.name + '의 친구: ' + friend);
            });
        }
    };

    person.printFriends();
    // 의 친구: 김철수
    // 의 친구: 이영희

    //Arrow Function 사용
    const person2 = {
        name: '홍길동',
        friends: ['김철수', '이영희'],
        printFriends: function() {
            this.friends.forEach(friend => {
                console.log(this.name + '의 친구: ' + friend);
            });

    person2.printFriends();
    // 홍길동의 친구: 김철수
    // 홍길동의 친구: 이영희
    ```
</details>

<details>
  <summary>(4) this 바인딩 메서드</summary>
  
  * **call - 즉시 호출**
    ```JavaScript
    function greet(greeting, punctuation) {
        console.log(greeting + ', ' + this.name + punctuation);
    }

    const person = { name: '홍길동' };

    greet.call(person, '안녕하세요', '!');
    // 안녕하세요, 홍길동!
    ```
    
  * **apply - 즉시 호출 & 배열로 인자 전달**
    ```JavaScript
    function greet(greeting, punctuation) {
        console.log(greeting + ', ' + this.name + punctuation);
    }

    const person = { name: '홍길동' };

    greet.apply(person, ['안녕하세요', '!']);
    // 안녕하세요, 홍길동!
    ```
    
  * **bind - this만 변경한 새 함수 반환(사용성이 가장 높은 함수)**
    ```JavaScript
    function greet() {
        console.log('안녕하세요, ' + this.name);
    }

    const person = { name: '홍길동', greet: function () { console.log('안녕하세요, ' + this.name); } };

    const boundGreet =  reet.bind(person);
    boundGreet();  // 안녕하세요 홍길동

    // 이벤트 리스터에서 유용
    button.addEventListener('click', person.greet.bins(person));
    ```  
</details>

<details>
  <summary>(5) this 정리</summary>

  * **this 결정 흐름**
    ```
    1. 일반 함수? -> 전역 객체
    2. 메서드 호출? -> 메서드를 호출한 객체
    3. call, apply, bind? -> 지정된 객체
    4. Arrow Function? -> 상위 스코프의 this
    5. new로 호출? -> 새 객체(인스턴스)
    ```
    
  * **실전 팁**
    ```JavaScript
    // 1. 메서드는 일반 함수로
    const obj = {
        method: function() {
            // this = obj
        }
    };
    
    // 2. 콜백은 Arrow Function으로
    obj.method = function() {
        setTimeout(() => {
            // this = obj (상위 스코프)
        }, 1000);
    };
    
    // 3. 이벤트 리스너는 bind
    button.addEventListener('click', obj.method.bind(obj));
    ```
</details>
