
# (3) 속성 접근
* **점 표기법**
```Javascript
let person = {
  name: '홍길동',
  age: 25,
  city: '서울'
};

console.log(person.name); // 홍길동
console.log(person.age); // 25
console.log(person.city); // 서울
```

* **대괄호 표기법**
```Javascript
let person = {
  name: '홍길동',
  age: 25
};

console.log(person['name']); // 홍길동
console.log(person['age']); // 25

// 변수 사용 가능
let key = 'name';
console.log(person[key]); // 홍길동
```

<br>

# (4) 속성 추가/수정/삭제
```Javascript
let person = {
  name: '홍길동',
  age: 25
};

// 속성 추가
person.city = '서울';
person.job = '개발자';
console.log(person); // {name: '홍길동', age: 25, city: '서울', job: '개발자'}

// 속성 수정
person.age = 26;
console.log(person.age); // 26

// 속성 삭제
delete person.job;
console.log(person); // {name: '홍길동', age: 26, city: '서울'}
```

# (5) 메서드(함수 속성)
```Javascript
let person = {
  name: '홍길동',
  age: 25,
  greet: function() {
    console.log(`안녕하세요!`);
  },
  introduce: function() {
    console.log('저는 ' + this.name + '입니다.');
  }
};

person.greet(); // 안녕하세요!
person.introduce(); // 저는 홍길동입니다.
```

<br>

# (6) 객체 순회
* **for...in문**
```Javascript
let person = {
  name: '홍길동',
  age: 25,
  city: '서울'
};

for (let key in person) {
  console.log(key + ': ' + person[key]);
}
// name: 홍길동
// age: 25
// city: 서울
```
